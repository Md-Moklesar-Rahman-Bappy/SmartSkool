# SmartSkool

School Management System - A lightweight system designed to efficiently handle notifications, messaging, friend requests, and user management within a school or educational platform.

---

## Features
- **Notifications**: Real-time alerts for important updates.
- **Messaging**: Direct communication between users.
- **Friend Requests**: Simple and secure social connections.
- **User Management**: Easy handling of accounts, roles, and permissions.

---

## Installation

1. Setup XAMPP (Apache + MySQL)
2. Create database `std_db` in phpMyAdmin
3. Import `mysql.sql` from root folder
4. Place project in `htdocs` folder
5. Access via `localhost/sms`

---

## Usage

Start Apache and MySQL in XAMPP, then visit `http://localhost/sms`
